# @four-bytes/opencode-token-budget-guard

**Status:** Sprint 1, geplant

Hook, der vor jedem LLM-Request die geschätzten Token zählt: Soft-Warning bei konfigurierbarem Threshold, Hard-Cancel bei Überschreitung. Diary-Eintrag mit Token-Statistik pro Session.

## Zweck

Kosten-Transparenz und -Kontrolle. Hello-World-Plugin um die opencode-Plugin-API zu lernen.
